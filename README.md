# zazametr

## Installation

1. Create `.env`-file in root dir. Check `.env.example`
2. Get [Docker](https://www.docker.com/products/docker-desktop/) and run

```bash
$ docker compose up -d
```

## Usage

```bash
# Stop running services
$ docker compose stop

# Start again
$ docker compose start

# Remove containers and volumes
$ docker compose down
```
